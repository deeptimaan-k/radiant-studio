💡 Best Practices for Every Project
How Radiant operates — standards we follow on every engagement
✅ What We Always Do
  • Register domain in the client's name
  • Show hosting cost as a separate line item
  • Take 50% advance before starting work
  • Provide a written scope agreement
  • Enable auto-renewal on domain
  • Give realistic, honest delivery timelines
❌ What We Never Do
  • Start work without any advance payment
  • Hide hosting costs inside the dev fee
  • Hold domain access hostage
  • Take large projects without a contract
  • Miss domain renewal dates
  • Add extra features without approval
💰 How Pricing Works
  • Hosting cost + 20% markup = what client pays
  • Developer fee is always one-time only
  • Domain is billed at actual cost, no markup
  • GST @18% added if client is GST-registered
  • Discounts available for long-term contracts
  • Referral discounts offered to existing clients
🚀 What You Get
  • Mobile-responsive design on all devices
  • Fast-loading, SEO-friendly structure
  • Complete source code & credentials handover
  • Step-by-step training on how to update content
